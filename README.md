## PR & Commit Convention
***PR Convention***
- `[Week][이름] - PR 내용` 으로 통일합니다.

```
ex) [Week1][승용] - 알고리즘 풀이
```

- 1주일 동안 해당 PR에 Commit을 쌓으시면 됩니다.

***Commit Convention***
- `[사이트-문제번호]: 문제 제목` 으로 통일합니다.
```
ex) [Baekjoon-1010]: 알파벳 찾기
```
